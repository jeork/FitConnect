<h1>FitConnect의 주요 기능</h1>

1. 전문적인 운동 가이드와 함께 개인 별로 루틴을 설계하고 기록할수 있는 기능

4. 정보 공유를 위한 커뮤니티 게시판과 유저간 메시지 및 채팅 기능
